The Shempa Core Plugin provides core functionality for<br>Shempa Wordpress Theme.


== Description ==

The Shempa Core Plugin allows for users of Shempa Wordpress Theme to benefit from a series of features that are at the basis of the theme.

= The features include =
* Custom Post Types: project and slide
* Custom Widgets: Social Widget, Flickr Widget, Latest Projects Widget, Image Hero Widget, Text Hero Widget
* Metaboxes with options for the different pages of the theme
* Translation Ready


== Installation ==

When you install Shempa Wordpress Theme, a message will appear in the Wordpress Dashboard that will allow you to install the plugin.
